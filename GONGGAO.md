# 📢 公告

## 2026-02-26 更新说明

本次版本重点解决了“看不到最新内容”的问题，并新增了课堂互动能力。

### ✅ 新增功能
- 🌐 多语言支持升级：新增 `Español`（西班牙语）和 `Français`（法语），语言按钮现在可循环切换 `中文 / English / Español / Français`。
- 🎮 新增课堂互动游戏：上线 `课堂互动游戏` 页面，支持随机题目、随机队伍、倒计时、即时加减分和历史记录。

### ✅ PWA 缓存更新（重点）
- Service Worker 升级到 `2.1.0`。
- 关键资源（`HTML/CSS/JS/JSON`）改为 **网络优先**，优先拉取最新版本，离线时再回退缓存。
- 媒体资源（图片/字体）继续使用缓存优化策略，兼顾性能与可用性。
- 版本号同步更新，触发客户端自动刷新与旧缓存淘汰，降低用户继续使用旧页面的概率。

### ✅ 仓库整理
- 首页资源版本号统一到 `2.1.0`。
- 新增页面：`src/pages/classroom-game.html`。
